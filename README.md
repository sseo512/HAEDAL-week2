###영화 소개  

+ __감독__ : 카일 볼다, 피에르 코팽
+ __개봉일__ : 2015년 7월 10일
+ __제작사__ : 일루미네이션 엔터테인먼트
+ __배급사__ : 유니버셜 픽쳐스  
+ __상영시간__ : 91분  
  
###개요  

<<__슈퍼배드__>>의 스핀오프 작품  
<<__슈퍼배드__>>의 프리퀄로서 미니언의 기원을 다룬다.  
  
발명가인 남편 허브와 함께 세상을 지배하려고 하는 악당 스칼렛 오버킬에게 미니언인 스튜어트, 케빈, 밥이 합세하면서 벌어지는 이야기이다.
